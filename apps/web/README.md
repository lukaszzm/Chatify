# @chatify/web

Frontend web application for Chatify, built with React, Vite, and TanStack Router.

## Features

- Real-time chat interface
- User authentication
- Note taking
- Mobile-friendly design

## Tech Stack

- React 19
- Vite
- TanStack Router
- URQL
- TailwindCSS
- TypeScript

## Getting Started

```bash
# Install dependencies
pnpm install

# Generate GraphQL types
pnpm codegen

# Start development server
pnpm dev
```
